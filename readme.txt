== Description ==
Download this interactive map of Florida WordPress plugin and customize the map colors, links, hover description for each of the 8 regions through an easy to use admin panel without any coding knowledge, then insert the map shortcode in any page/post to display the map.

Are you looking for [Premium Interactive Map of Florida](https://www.wpmapplugins.com/states/interactive-map-of-florida-wordpress-plugin.html) with clickable counties?

Features:

*	Interactive map of Florida with 8 clickable regions.
*	Does NOT require ANY coding knowledge.
*	Customize each region independently, so you can change colors, link, hover information for each region.
*	Easy to use admin panel interface.
*	Bulk edit option.
*	Works perfectly on desktop along with mobile devices including Smartphones: iPhone, iPad, Tablets, etc.
*	Install and customize the map in minutes then embed the shortcode in any page or post to display the map.

You may be also interested in another [state map with clickable counties](https://www.wpmapplugins.com/states/individual-us-state-maps-package-wordpress-plugins.html).

== Installation ==

1. Install the plugin .zip file through your WordPress 'Plugins' screen directly, or Upload the plugin files to the `/wp-content/plugins/interactive-map-of-florida` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Now, you will see a new menu button in the left side bar of your WordPress called 'Florida Reg. Map', click it and customize your map then save the changes.
4. Use this shortcode `[flr_map]` to display the map in any page or post.

== Frequently Asked Questions ==

= Can I add an image in the hover popup? =

Yes, you can write any text and add images.

= Does this map requires Flash to work? =

No, it's SVG based and works perfectly on desktop along with mobile devices 'smartphone'.

= Is this map responsive? =

Yes.

= Can I split region into counties? =

No, but fortunately we already have a [Premium Interactive Map of Florida](https://www.wpmapplugins.com/states/interactive-map-of-florida-wordpress-plugin.html) with clickable counties.

= Can I set a region such as Northwest Florida Panhandle as inactive? =

Yes, simply un-check the (active) box beside that region to make it inactive.

= Can I use the shortcode in different pages to display the same map? =

Yes.

= Can I create a different map and use it in another page? =

No, but you can buy the **[Multiple Instances Regional Map of Florida Package](http://wpflr.html5maps.pay.clickbank.net/)** to get 3 instances of the plugin and install each one to get a different styled map.

= Can I link a region to any webpage? = 

Yes, also you can choose to open it in the same window or in a new window.

Yes, we can customize the map for what you need as a freelance customization service, just [contact](https://www.wpmapplugins.com/states/interactive-map-of-florida-wordpress-plugin.html#contact) us.
